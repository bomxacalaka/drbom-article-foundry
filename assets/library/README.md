# Article Asset Library

Place reusable article assets here. When creating a new article, inspect this folder first and only copy assets that fit the article.

Raw one-off user uploads should start in `assets/inbox/`. Move reusable assets here only when they are likely to help future articles.

Recommended formats:

- thumbnails: `.webp`, `.avif`, or optimized `.png`
- diagrams/icons: `.svg` when vector is appropriate
- video: compressed `.mp4`
- 3D: `.glb` or `.gltf` with compressed textures where possible

Avoid placing credentials, private files, or unlicensed media in this folder.
