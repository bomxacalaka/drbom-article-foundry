Apr 16, 2026 system demo

This first article exists to prove the publishing path works: static files on S3, live likes and comments through Lambda, and article metadata generated into the landing page.

What this validates

The article page is plain HTML, CSS, and JavaScript. The browser reads public static files from the `articles/` prefix and uses the shared shell script to load counts, post likes, and submit comments.

How future articles are made

Each new article gets its own folder, metadata file, stylesheet, script, and assets directory. The index generator scans those folders and publishes only articles that are not marked as drafts.
